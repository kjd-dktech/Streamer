📺 Streamer

Streamer vous permet de diffuser votre écran vers une TV ou un appareil local, comme un câble HDMI sans fil.

🔧 Installation (RPM)

Téléchargez puis installez :

📦 Télécharger streamer.rpm

Ensuite dans un terminal :

sudo dnf install ./streamer-1.0-1.noarch.rpm

▶️ Utilisation

Recherchez “Streamer” dans votre menu GNOME et suivez les instructions à l’écran.
